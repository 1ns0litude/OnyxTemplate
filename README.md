
# 🖤 OnyxTemplate

**OnyxTemplate** is a CLI tool that automatically generates structured, Obsidian-friendly note-taking environments for:

- 🧩 Capture The Flag Challenges [CTF MODE]
- 🌍 Real-World hacking engagements [WWW MODE]

Designed for hackers who want **speed, consistency, and clean markdown**.

---

## ✨ Features

- Full CLI interface
- Two modes:
  - **CTF (Capture The Flag) Mode** – fast, challenge-oriented layout
  - **WWW (Wild Wild West) Mode** – real-world engagement structure
- Obsidian-optimized Markdown templates
- Works with any Markdown-based note system
- No dependencies on Obsidian itself
